# Release Plan — Icon Library

## Product
**L7 Icon Library — 119 Original Icons**
**Creator:** Alberto Valido Delgado

---

## Release Strategy

### Phase 1: Bundled (Current)
- Included with L7 framework installation
- Icons used throughout L7 Universal OS, Emporium, and all tools
- PNG and ICNS formats

### Phase 2: Icon Pack Release (Q2 2026)
- Standalone icon pack available at avli.cloud/icons
- SVG versions for web use (vector conversion)
- Multiple sizes: 16px, 32px, 64px, 128px, 256px, 512px, 1024px
- Preview gallery with search and filtering

### Phase 3: Design System (Q3-Q4 2026)
- Figma component library
- Sketch symbol library
- Icon font (woff2) for web embedding
- CSS sprite sheets
- Design guidelines document (spacing, color, usage rules)

### Phase 4: Expanded Collection (2027)
- Additional icons for community-requested categories
- Animated icon variants (Lottie/CSS animations)
- Dark mode variants
- Platform-specific adaptations (iOS, Android, Windows)
- Licensing for third-party products

## Distribution Channels
- L7 Framework: Bundled (default)
- avli.cloud/icons: Standalone purchase and download
- Figma Community: Design system library (future)
- GitHub: Source files (L7 repository)
