# Release Plan — Living Rose

## Product
**Living Rose — Prima Language Visualization**
**Creator:** Alberto Valido Delgado

---

## Release Strategy

### Phase 1: Direct Download (Current)
- Available as compiled binary in L7 repository
- Launched via shell script or direct execution
- macOS arm64 only

### Phase 2: Polished Release (Q2 2026)
- Code-signed with Apple Developer certificate
- Notarized for Gatekeeper approval
- DMG installer with drag-to-Applications
- Direct download from avli.cloud/rose

### Phase 3: App Store (Q3 2026)
- Submit to macOS App Store ($4.99)
- App Store screenshots and description
- Sandboxed version for App Store compliance
- Free companion download also available from avli.cloud

### Phase 4: Cross-Platform (2027)
- iPad version (SwiftUI port)
- Web version (standalone HTML5, no server required)
- Integration with Prima compiler for live sigil rendering

## Distribution Channels
- avli.cloud: Direct download (free)
- macOS App Store: Paid version ($4.99)
- GitHub: Source code (L7 repository)
- Academic: Supplement to Prima language paper
