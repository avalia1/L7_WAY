A visual interface for the Prima language Rose Cross. The Living Rose renders the 22 operations of Prima as an interactive visualization, mapping Hebrew letters and Tarot Major Arcana onto a navigable rose cross diagram. The user can explore operations, connections, and weighted hypergraph relationships through direct manipulation.

Built entirely in Swift 6.2.4 with no foreign dependencies. Uses only Apple system frameworks (AppKit, WebKit). The binary is a self-contained Mach-O arm64 executable.
