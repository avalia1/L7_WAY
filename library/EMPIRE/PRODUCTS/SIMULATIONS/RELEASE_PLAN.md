# Release Plan — Simulations

## Product
**Interactive HTML5 Simulations**
**Creator:** Alberto Valido Delgado

---

## Release Strategy

### Phase 1: Academic Supplements (Current)
- Available in L7 repository
- Supplements to IQS-888 paper and L7 architecture papers
- Self-contained HTML files, no installation required

### Phase 2: Online Gallery (Q2 2026)
- Host at avli.cloud/simulations
- Interactive gallery with descriptions and context
- Embeddable iframe versions for academic papers
- Mobile-responsive touch controls

### Phase 3: Exhibition Ready (Q3-Q4 2026)
- Museum/gallery installation packages
- Kiosk mode (fullscreen, no browser chrome)
- Multi-display configurations
- Printed companion materials (descriptions, QR codes)
- Licensing for permanent and temporary exhibitions

### Phase 4: Educational Platform (2027)
- Interactive lesson plans built around simulations
- Student exercises with modifiable parameters
- Teacher dashboard for classroom use
- New simulations added quarterly
- Educational institution licensing

## Distribution Channels
- GitHub: Source code and self-contained HTML files
- avli.cloud/simulations: Online gallery
- Academic papers: Embedded supplements (arXiv, journal submissions)
- Museums/Galleries: Exhibition licensing
- Educational: Institutional licensing
