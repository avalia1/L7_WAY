## Originality Declaration
Prima is an original programming language. While it draws inspiration from Kabbalistic, alchemical, astrological, and Tarot traditions (which are public domain cultural heritage), the mapping of these systems into a formal programming language with compilation to weighted hypergraphs has no prior art. The concept of sigils as compilation targets — weighted graphs where edges carry 12-dimensional semantic weights — is an original contribution to programming language design.
