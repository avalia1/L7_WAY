# Release Plan — Prima Language

## Product
**Prima — The Programming Language of the Forge**
**Creator:** Alberto Valido Delgado

---

## Release Strategy

### Phase 1: Internal (Current)
- Compiler operational within L7 framework
- Specification documented in FOUNDERS_DRAFT.md and ARCHITECTURE_FULL.md
- Living Rose provides visual interface

### Phase 2: Academic Publication (Q2 2026)
- Publish language specification as academic paper
- Submit to programming language conferences (SPLASH, OOPSLA, or art/code venues)
- Release specification document under Creative Commons
- Open source reference compiler (lib/prima.js) under L7 license

### Phase 3: Developer Tools (Q3-Q4 2026)
- Standalone Prima REPL (read-eval-print loop)
- VS Code / editor syntax highlighting
- Prima playground on avli.cloud
- Tutorial series: "Learning to Speak Prima"

### Phase 4: Ecosystem (2027)
- Prima package manager (sigil registry)
- Standard library of common sigil patterns
- Integration with other graph databases and visualization tools
- Community-contributed operation extensions

## Distribution Channels
- Academic: Conference papers, arXiv preprint
- GitHub: Open source specification and reference compiler
- avli.cloud: Interactive playground and documentation
- Living Rose: Visual companion application
