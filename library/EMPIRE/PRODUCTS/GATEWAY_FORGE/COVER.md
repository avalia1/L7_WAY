The Gateway is the Unified Self of the L7 system. Every tool invocation, every data flow, every transformation passes through it. The Gateway is not a router — it is an orchestrator that understands context, intent, and the dimensional coordinates of every citizen in the empire.

The Forge is the transmutation engine. It implements the four-stage alchemical process:
1. **Nigredo** — Decomposition (break input into raw elements)
2. **Albedo** — Purification (clean, validate, normalize)
3. **Citrinitas** — Illumination (enrich, analyze, understand)
4. **Rubedo** — Integration (synthesize into final output)

Together, Gateway + Forge are the heart of L7. The Gateway decides what must be done; the Forge decides how to transform it. Everything else in the system serves these two.
